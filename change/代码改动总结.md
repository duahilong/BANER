# BANER 项目代码改动总结

## 改动概述

本次改动将 BANER 项目从 LLaMA-2-7B 模型切换到 Qwen1.5-7B 模型，以支持中文 NER（命名实体识别）任务。

**改动的文件**:
- `myfinetune.py` - 训练脚本
- `myinference.py` - 推理脚本
- `generate.py` - 生成脚本（Gradio 界面）
- `templates/qwen.json` - 新增中文提示词模板
- `.gitignore` - 新增 Git 忽略文件配置

**改动的行数**: 140 行插入，31 行删除

---

## 改动原因

### 原有模型（LLaMA-2-7B）的局限性

1. **训练语言**: 主要为英文，中文数据极少
2. **词表支持**: 几乎没有中文 token
3. **中文理解**: 极差，无法正确处理中文文本
4. **分词效率**: 中文会被拆分成大量字节级 token，效率极低

### Qwen1.5-7B 的优势

1. **中英双语**: 完整支持中文和英文
2. **词表完整**: 包含完整的中文词表
3. **理解能力强**: 对中文的理解和生成能力优秀
4. **分词高效**: 中文词级分词，效率高

## 使用说明

### 1. 下载模型

```bash
# 使用 huggingface-cli 下载
huggingface-cli download Qwen/Qwen1.5-7B --local-dir ./models/qwen1.5-7b

# 或使用 modelscope 下载
pip install modelscope
python -c "from modelscope import snapshot_download; snapshot_download('qwen/Qwen1.5-7B', cache_dir='./models')"
```

### 2. 训练模型

```bash
python myfinetune.py \
    --base_model './models/qwen1.5-7b' \
    --data_path './data/GUM/train.json' \
    --output_dir 'lora_finetune_qwen' \
    --prompt_template_name 'qwen'
```

### 3. 推理测试

```bash
python myinference.py lora_finetune_qwen
```

### 4. 启动 Gradio 界面

```bash
python generate.py --base_model './models/qwen1.5-7b' --lora_weights 'lora_finetune_qwen'
```
